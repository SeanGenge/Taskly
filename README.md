# Taskly 

## Description
A simple Todo app that allows you to add or delete tasks.

The project uses Angular 20 and ASP.NET core web API with .NET 9.0 Framework.

For completeness, I included the instructions down below.

## Installation
To install the necessary dependencies, run the following command:

1. Open command prompt
2. cd into taskly.web
3. Type npm i

## To run the project
1. Open taskly.api in Visual Studio
2. Start the project
3. Open taskly.web in command prompt
4. type 'npm start' and press enter
5. go to http://localhost:4200/

## To run tests
1. Open taskly.api in Visual Studio
2. Click on the test tab
3. Click on Run all tests
